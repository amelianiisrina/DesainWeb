Website ini sekarang merupakan PWA.

Artinya, kamu dapat:

  • Akses offline: tetap bisa membuka website tanpa internet
  • Instalasi di perangkat: kamu dapat menginstal website di layar utama seperti aplikasi biasa.
  • Load lebih cepat: data tersimpan di cache, jadi buka halaman lebih cepat.

Singkatnya, pengalaman browsing jadi lebih nyaman dan mirip aplikasi asli di perangkat kamu!
